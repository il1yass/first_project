Этот проект для экзамена представляет собой веб-приложение о музыкальных жанрах, исполнителях и их биографиях. Он основан на Django и использует базу данных SQLite 3 для хранения информации.

Функциональность

  1. Жанры музыки: Предоставление информации о различных музыкальных жанрах.
  2. Исполнители: Подробная информация о различных музыкальных исполнителях.
  3. Шаблоны: Использование шаблонов для создания пользовательского интерфейса.

Основные технологии

  • Django: Фреймворк Python для создания веб-приложений.
  • SQLite 3: Легковесная реляционная база данных.
  • Шаблоны: Использование шаблонов для отображения данных.

Возможные развития

  • Добавление пользовательских профилей.
  • Расширенный поиск по музыкальным жанрам и исполнителям.
  • Административная панель для управления контентом.
